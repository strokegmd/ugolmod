# UGOLMod
крутой мод чтобы уголь крч был!! (в майне)<br>
че может уголь:
- строить пенисы из беброка
- строить кресты из беброка
- уголь крашит майн если подойти к нему минимум на 5 блоков
- поджигать и убирать листву
